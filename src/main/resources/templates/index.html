Molokotech OAuth 2.0 Implementation

What is this?

This is an implementation of OAuth 2.0 protocol, it is a new project with a server side engine acting like a filter for the Client Apps configured to be secured. To resume, is an universal server side filter to secure all Molokotech Apps or other ones that wanna to be secured from us. There is no pages to navigate apart of this one, all the code is on the back side.

Most popular providers of this kind of secure system are: Facebook, Google, Twitter, Instagram, Github, Okta and anyone else.
